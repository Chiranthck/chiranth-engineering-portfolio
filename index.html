Project Images

Project Overview

Designed and constructed a comprehensive mock layout of the Maruti 800 car electrical system, demonstrating the interconnection of key automotive components including battery, alternator, starter motor, lighting systems, and control modules.

Key Components Implemented

  • Power Generation & Storage: Battery, Generator Assembly, Alternator
  • Engine Systems: Starter Motor Assembly, Distributor, Spark Plug
  • Lighting Systems: Headlights, Tail Lamps, Indicators, Interior Lighting
  • Control Systems: Combination Switch, Steering Lock, Relays
  • Monitoring: Fuel Gauge Assembly, Multi-meter
  • Safety Systems: Horn, Wiper Motor

Taguchi-Grey Relational Analysis for Radish Yield Optimization

Project Images

Project Overview

Applied Design of Experiments (DOE) methodology using Taguchi's Orthogonal Arrays and Grey Relational Analysis to systematically optimize radish cultivation.

Key Results & Achievements

  • Optimal Combination: Trusted Seeds + Drip Irrigation + DAP Manure
  • Average Length: 14.75 cm
  • Quantity: 35.5 units
  • Weight: 2.25 kg
  • Grey Relational Grade: 0.8397 (highest among all treatments)
